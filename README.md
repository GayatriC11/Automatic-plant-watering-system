Smart watering system is my personal project built on real life application for understanding circuit designing and sensors usage. 

Designed and simulated an automatic watering system using Arduino, soil moisture sensor, LCD, buzzer, LED, servo motor, and DC motor.
Displayed soil moisture levels on an LCD screen for monitoring.

Used a buzzer and red LED to indicate low moisture conditions.

Programmed a servo motor to automatically open the water valve when soil moisture was low and close it when moisture levels became adequate.
Controlled the movement of the vehicle using a DC motor, allowing it to move forward and stop when low moisture areas were detected and used ultrasonic sensor for obstacles.

Tested and verified the complete system through Tinkercad simulation.
